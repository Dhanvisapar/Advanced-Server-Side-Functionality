# Advanced-Server-Side-Functionality

## Objective
To implement advanced server-side features including middleware, background tasks, and caching.

## Features Implemented
- Logging and body-parsing middleware
- Background job processing using task queue
- Server-side caching for performance optimization

## Endpoints
- POST /task → Add background job
- GET /data → Cached server response

## How to Run
1. Open folder in VS Code
2. Run:
   npm install
   npm start
3. Server runs at:
   http://localhost:3000

## Learning Outcomes
- Middleware usage
- Background task processing
- Caching strategies
